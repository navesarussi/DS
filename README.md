# ScaleDigital

אתר השיווק של ScaleDigital - פתרונות דיגיטליים לעסקים קטנים ובינוניים.
נבנה עם Next.js, TypeScript ו-Tailwind CSS.

## פיתוח מקומי

```bash
npm install
npm run dev
```

האתר יעלה בכתובת [http://localhost:3000](http://localhost:3000).

## עריכת תוכן

- **פרטי קשר, קישורי פרויקטים ופרטי המייסדים**: `src/lib/site-config.ts`
- **סקשנים באתר**: `src/components/`

## פריסה

הפרויקט מוכן לפריסה ב-[Vercel](https://vercel.com/new) - מספיק לחבר את ריפו ה-GitHub הזה ולפרוס.

```bash
npm run build
```
